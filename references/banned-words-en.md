# Banned Words, Phrases, and Openers — English

These are statistically flagged as AI-generated text markers across multiple studies (Carnegie Mellon 2025, Wikipedia Signs of AI Writing, Buffer 52M post analysis, community detection patterns from X/Reddit). Never use any of these. Replace with concrete alternatives or restructure the sentence.

## Model-Specific First-Word Tells (avoid starting responses with these)

Gemini tends to start with: "my," "creating," "while," "here," "yes," "this," "the"

## Replacement Strategies

Instead of reaching for a banned word, try:

| Instead of... | Try... |
|---|---|
| delve into | look at, dig into, examine, pick apart |
| comprehensive | thorough, covers [X and Y], full |
| leverage | use, take advantage of, build on |
| utilize | use |
| facilitate | help, make possible, run |
| seamless | smooth, easy, painless |
| robust | strong, solid, reliable |
| innovative | new, different, [describe what's actually new] |
| transformative | changed [what, specifically] |
| unprecedented | first, new, rare, [describe the actual rarity] |
| it's worth noting | [just state the thing] |
| at the end of the day | [just state the conclusion] |
